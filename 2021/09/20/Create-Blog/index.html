Create Blog

npm install -g cnpm –registry=http://registry.npm.taobao.org

git的一些准备工作

ssh-keygen -t rsa -C “15697891287@163.com“

git config –global user.name “XXXXn”
git config –global user.email “XXXX7@163.com”

cnpm install -g hexo-cli

新建一个文件夹，如果处理问题就将这个文件夹删除，重来

hexo init

启动预览： hexo s

产生一篇新文章： hexo n “第一篇文章”
进入blog/source/_posts/ 下编写自己的博客

之后，退回到 blog/ 目录下

hexo clean
hexo g 生成一下
hexo s 预览一下

上面的操作都是在本地操作，后面来部署在github里

在github里新建一个项目，项目名必须是 用户名.github.io

LIUhansen.github.io

回到命令行
cnpm install –save hexo-deployer-git

打开配置文件 .blog/_config.yml

最后一行，： 后面需要一个空格
type: git
repo: https://github.com/LIUhansen/LIUhansen.github.io.git
branch: master

出现问题，有的博客说上面的写法是错的，需要改成

git@github.com:LIUhansen/LIUhansen.github.io.git

github里面有了更新，但是
LIUhansen.github.io.git网址并没有我的内容

部署到远端：

hexo g
hexo d 更新你修改的地方

添加主题：

github.com/litten/
https://github.com/litten/hexo-theme-yilia.git
hexo-eheme-yilia

修改配置文件
倒数第5行
theme: yilia

hexo clean
hexo g
hexo s
需要等一会才能生效
